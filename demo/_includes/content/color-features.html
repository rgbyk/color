About

This project started with research into visual disabilities and web accessibility issues for a design system I've been developing since 2013. It's become the color system component of that application.

The library is packed with features to analyze color temperature, contrast, and automate design elements for your projects.

Cases

With this library, you can:

  • Build a fully documented color system for seamless integration into any web or app project.
  • Create utility classes and choose your output format: CSS custom properties or HEX codes.
  • Easily toggle between RGB and RYB color models.
  • Design color relationships and achieve a harmonious blend within your color schemes.
  • Neutralize colors to create a balanced, neutral tone that complements chromatic hues.
  • Match color temperatures across your grayscale palette for consistency.

Features

  • Switch color palettes on or off, including Complementary, Analogous, Square, Split, Triadic, and Tetradic options.
  • Enable or disable web accessibility features for color blindness, covering Protanopia, Protanomaly, Deuteranopia, Deuteranomaly, Tritanopia, Tritanomaly, Achromatopsia, and Achromatomaly.
  • Toggle high-contrast colors on or off to ensure proper contrast is applied in reverse order.
  • Design an "alert palette" for UI components to indicate status like error, warning, or success.
  • Turn utility classes on or off for background, border, and text colors, using .bg-*, .border-*, and .color-* classes.
