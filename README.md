# @rgbyk/color

## requirements
- [node](https://nodejs.org/) v12.11.0
- [sass](https://sass-lang.com/dart-sass) v1.26.0 (compiled with dart2js 2.7.1)

----

## getting started

### installation

1. `npm run unpack-rgbyk`

### cli

1. `npm run color` 
2. `npm run color-rgb`
2. `npm run color-ryb`

----
